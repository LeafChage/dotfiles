read `_claude-notes/README.md` in the CoffeeBeagle_ Obsidian vault, then
the files it links to. These notes describe how Kusanagi thinks, works,
and prefers to collaborate — apply them regardless of which project this
session is in.

When a genuinely new, recurring, or corrected pattern about how Kusanagi
works with Claude shows up in *any* project (not just the vault itself),
update the relevant file in `_claude-notes/` the same way you would from
within the vault.
